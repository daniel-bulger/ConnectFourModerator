About this program...

This program was created by Daniel Bulger and Derek Schultz for use in the Rennselaer Polytechnic Institute's Minds and Machines class. It is a cross-platform alternative to the existing Connect Four moderator program.

Source and installation packages for Windows, Mac, and Linux can be found at the project's Github repository:
https://github.com/Moomaster36/ConnectFourModerator
